# Dogs

## サイト概要
### サイトテーマ
自分の飼っているワンちゃんを発信し、犬好きと交流を深められるSNSサイト

### テーマを選んだ理由
もともと犬が好きで、犬を一匹飼っているのですが、他の人がどのような犬を飼っているのか、どのような育て方をしているのかが知れるサービスがあれば便利だと考え、このテーマにしました。

### ターゲットユーザ
犬が好きな方（ペットを飼っていなくても利用可能）

### 主な利用シーン
気になる犬の投稿を検索したり、自分の飼っているペットの様子を投稿したりできる
